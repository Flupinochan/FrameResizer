# FrameResizer
A tool to batch resize and add borders to multiple images.

複数の画像を一括でリサイズ、枠線を付けるツールです!

ブログで画像を投稿する際、運用手順書に画像を貼り付ける際に有用です
- 横幅が統一されている画像 => 見やすい!
- 枠線がついている画像 => 見やすい!
